Setup

Choose your preferred way to get started with the examples.

Option 1: Web UI

  1. Open the Web UI at http://localhost
  2. Navigate to File > Import and select an example YAML file
  3. Click Run — results appear in real time

Option 2: CLI

# Run a single test suite
npx api-automation run examples/01-basic-crud.yaml

# Run all examples
npx api-automation run examples/

# Run with a tag filter
npx api-automation run examples/ --tags smoke
